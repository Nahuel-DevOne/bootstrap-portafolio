# Portafolio en Bootstrap 5

### * Tiene de todas maneras instalado SASS, y clases de CSS, para darle retoques personalizados a los estilos típicos de Bootstrap.
### * Tiene un pequeño trabajo de JavaScript, para darle el efecto a las imágenes del portafolio, de que se agranden cuando se les haga click.
